Add an API connection

Store the API ID and API key for a job search provider. Nothing leaves this device except the requests LaunchPad makes on your behalf when you click Refresh.

Adzuna is free and covers many countries with one account.
  1. Go to developer.adzuna.com and register — no card needed.
  2. Your account page shows an Application ID and an Application Key.
  3. Pick the country you want to search, then paste both values below.
Header-based auth only works if the API allows direct requests from a browser (CORS). If Refresh fails to connect, the API likely blocks that — try Query string mode if the provider supports it, or check the provider's CORS policy.
⚙ Advanced: response field mapping ▾

Tell LaunchPad where to find each field in the JSON response. Dot notation is supported (e.g. company.display_name). Defaults match Adzuna-style APIs.
